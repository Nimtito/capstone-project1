KABZ  BAKERY WEBSITE

## Description
Kabz smart bakery is a simple and responsive website built to help a local bakery showcase its products and receive customers online

# Problem Statement
Currently Kabz smart bakery advertises products on whatsapp status ,customers make orders via whatsapp or phone calls and there is no online presence that is a website and customers give reviews

# Solution
Build a simple and responsive website that has images of the products, has order page where customers can feed their information and make orders of the product they want,pick up date or order on delivery,create a contact page with the location ,whatsapp link and phone number.


# Objectives
* To show images of the available products
* To make customers view the menu and make orders
* To simplify ordering process by using either phone number or whatsapp


# Technologies Used
 * HTML
 * CSS
 * Tailwind CSS
